# Blackboard All Students Selector Extension
## إضافة اختيار جميع الطلاب في البلاك بورد

هذه إضافة لمتصفح Chrome/Edge تضيف خيار "All Students" في صفحة إرسال الرسائل بالبلاك بورد.

## المميزات
- ✅ إضافة خيار "All Students" تحت "All instructors"
- ✅ اختيار جميع الطلاب فقط (الذين معرفاتهم تبدأ بـ S متبوعة بأرقام)
- ✅ استبعاد المدرسين والمشرفين تلقائياً

## طريقة التثبيت

### Google Chrome:
1. افتح Chrome واذهب إلى: `chrome://extensions/`
2. فعّل "وضع المطور" (Developer mode) من أعلى الصفحة
3. اضغط على "تحميل غير مُحزّم" (Load unpacked)
4. اختر مجلد `send_message`
5. الإضافة جاهزة للاستخدام!

### Microsoft Edge:
1. افتح Edge واذهب إلى: `edge://extensions/`
2. فعّل "وضع المطور" (Developer mode)
3. اضغط على "تحميل غير مُحزّم" (Load unpacked)
4. اختر مجلد `send_message`
5. الإضافة جاهزة!

## طريقة الاستخدام
1. اذهب إلى صفحة إرسال الرسائل في البلاك بورد
2. اضغط على حقل "Recipients"
3. ستجد خيار "All Students" تحت "All instructors"
4. اضغط عليه لاختيار جميع الطلاب تلقائياً

## ملاحظات
- الإضافة تختار فقط المستخدمين الذين معرفاتهم تبدأ بـ `s` متبوعة بأرقام (مثل : s23**)
- لا يتم اختيار المدرسين أو المشرفين

## الملفات
- `manifest.json` - إعدادات الإضافة
- `content.js` - الكود الرئيسي
- `styles.css` - التنسيقات
- `icon.svg` - أيقونة الإضافة
